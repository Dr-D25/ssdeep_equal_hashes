# ssdeep_equal_hashes
Веб-приложение для поиска похожих файлов по нечётким хэшам ssdeep с поддержкой базы вредоносных сигнатур, кластеризации и наглядной подсветки совпадающих участков.

# Основные возможности
- Попарное сравнение файлов – поиск похожих файлов с заданным порогом схожести.

- Кластеризация – автоматическая группировка файлов по схожести (алгоритм DSU).

- Встроенная база вредоносных хэшей – сохраняйте образцы вредоносного ПО с тегами, затем сканируйте новые файлы на предмет совпадений.

- Умная подсветка – все совпадающие фрагменты в блоках ssdeep выделяются красным цветом (на основе LCS), что упрощает визуальный анализ.

- Работа с ZIP-архивами – загружайте архивы, они будут автоматически распакованы и обработаны рекурсивно.

- Экспорт CSV – результаты сравнения и сканирования можно сохранить для дальнейшего анализа.

# Как это работает
## Нечёткие хэши ssdeep
ssdeep (известный как Context Triggered Piecewise Hashing) позволяет находить файлы, которые похожи, но не идентичны. Это идеально подходит для обнаружения модифицированных вредоносных программ, изменённых документов и т.п.

## База данных
Все добавленные хэши сохраняются в локальную SQLite-базу (ssdeep_hashes.db). Для каждого файла можно указать тег – например, название вредоноса, семейство, версию. При сканировании новые файлы сравниваются со всей базой по порогу схожести, и выдаются все совпадения с указанием тега и процента.

## Подсветка совпадений
Функция подсветки использует алгоритм поиска наибольшей общей подпоследовательности (LCS), поэтому выделяются все общие участки, даже если они смещены. Это делает сравнение гораздо более информативным, чем просто позиционное сравнение.


# Интерфейс и режимы работы
Приложение разделено на 4 вкладки:

### 1. 📂 Управление базой
- Добавление файлов – загрузите файлы или ZIP, укажите тег (например, Trojan.Generic.123) – они сохранятся в базу.

- Просмотр – таблица всех записей с возможностью удаления по ID.

### 2. 🛡️ Сравнение с базой
- Загрузите подозрительные файлы.

- Установите порог схожести (по умолчанию 70%).

- Получите список совпадений с указанием тега, процента и подсветкой совпадающих блоков.

- Для каждого совпадения можно раскрыть детали с выделением общих фрагментов.

### 3. 🔗 Попарное сравнение
- Сравните несколько файлов между собой.

- Настройте порог, показ хэшей, блоков, подсветку.

- Результаты сортируются по убыванию схожести.

- Можно скачать CSV.

### 4. 📊 Кластеризация
- Автоматическое группирование файлов в кластеры по схожести.

- Задайте минимальный размер кластера.

- Внутри кластера отображаются попарные сравнения с подсветкой (если включена).
